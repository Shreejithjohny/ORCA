Architecture & Modularity Rules7.1  The One-Direction RuleData flows in one direction only:Collection  ‚Üí  Redis  ‚Üí  Agents  ‚Üí  Correlator  ‚Üí  FastAPI  ‚Üí  FrontendNo layer may call a layer above it. The only exceptions are: Member A's Neo4j, which FastAPI may read, and Ollama, which the NLP endpoint calls internally.7.2  Module BoundariesEach AI agent is a standalone asyncio process with its own Kubernetes Deployment. Agents must not import from each other.The Correlator is the only service that writes to incidents:* streams. Agents write only to anomalies:*.FastAPI is the only service that serves HTTP to the frontend. The frontend never connects to Redis, Neo4j, or Ollama directly.The NLP chain (chain.py) is imported by FastAPI as a module ‚ÄĒ it is not a separate microservice. Keep it stateless.7.3  Environment Variables ContractVariableFormat / ExampleUsed ByREDIS_URLredis://orca-redis.orca.svc.cluster.local:6379/0All agents + FastAPINEO4J_URIbolt://orca-neo4j.orca.svc.cluster.local:7687FastAPI + CorrelatorNEO4J_USER / NEO4J_PASSWORDKubernetes Secret neo4j-authFastAPI + CorrelatorOLLAMA_HOSThttp://ollama.orca.svc.cluster.local:11434FastAPI (NLP endpoint)VITE_API_BASE_URLhttp://localhost:8080 (local dev)React frontend8.
